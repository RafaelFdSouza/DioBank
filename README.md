Fixar e praticar os princípios deste paradigma.<br><br>
<strong>4 pilares da orientação a objetos:</strong>
<ul>
<li>Encapsulamento</li>
<li>Abstração</li>
<li>Herança</li>
<li>Poliformismo</li>
